使用Flask-Migrate实现数据库迁移

安装：pip install flask-migrate

配置：

  1. 第一次使用的时候需要初始化迁移仓库。python manage.py db init
  2. 创建迁移脚本。python manage.py db migrate -m "migration message"
  3. 更新数据库。python manage.py db upgrade
