账户和密码从 Chrome 导入到 Opera

Chrome Opera
 2018/12/29

放弃使用 Chrome 转向 Opera 纯粹是用腻了。在 Chrome、Opera、Firefox 的性能对比中，Opera 是最接近 Chrome 的，并且界面功能也能给我带来些新鲜感，于是乎就选择了它。

从一个浏览器切换到另一个浏览器唯一需要操心的是个人数据的迁移问题，现代化的浏览器都明显地提供了方便的书签、Cookie、浏览历史的迁移功能。但是网站的账户密码的迁移却没那么直接。

迁移网站的账户和密码

开启 Opera 导入功能

在 Opera 浏览器的地址栏输入 opera://flags/#PasswordImport ，然后将 Password import 选项设置为 Enabled 。这样就打开了浏览器的密码导入功能。

导出 Chrome 密码为 CSV 文件

打开 Chrome 浏览器，在地址栏输入 chrome://settings/passwords 进入密码管理页面，点击导出密码，将密码以 CSV 格式存储在电脑磁盘上。

导入密码到 Opera

在 Opera 浏览器的地址栏输入 opera://settings/passwords ，点击导入密码的选项，并选择上一步导出的 CSV 文件进行导入。

完成以上步骤即完成了账户和密码的迁移了。
